Nimeni on Rustam

Hei tuleva työnantaja!

👋
Kirjoitin vähän itsestäni, olisin kiitollinen, jos luisit loppuun asti.😉
✅ Minä on ollut töissä monilla aloilla: myynnissä, kirjanpidossa, laskennassa, ehdin olla yrittäjänä (omistin kahvilan). Sitten vuoden 2016 alussa muuttin Suomeen🇫🇮perheeni kanssa. Olin töissä ateriapalvelujen alalla laivoilla - Viking-line, Silja-line, Finnlines. Pandemian aikana minulla oli paljon vapaa-aikaa ajattella tulevaisuudesta ja arvioida työtilanne uudelleen. Olen ollut kiinnostunut ohjelmoinnista koko elämäni aikana ja joulukuusta 2020 lähtien💻 alloitin opiskelu käyttöliittymäkehityksellä, HTML: llä ja CSS: llä. 🔥Siitä hetkestä lähtien innostuin kovasti ja löysin hyvän verkkokurssin Udemystä. Nautin prosessista ja tuloksesta, mitä vaikeampi tehtävä on, sitä mielenkiintoisempi se on. Lisätavoitteena on hallita Java Script (oppiminen nyt prosessissa) ja suositut kirjastot.

✅Tällä hetkellä suoritanut Veb-kehityksen verkkokurssit: rakennussivustot, html, css, jquery, Bootstrap, gulp, Sass / Scss-esiprosessorit ja muut, Udemyssa. Myös oppin Java Script, samassa paikassa Udemyssä.

✅Mitä olet oppinut? Opinut luomaan sivustoja asetteluista Figma, html, css, työskentellä kirjaston kanssa Bootstrap, sekä muokata ja työskentellä esiprosessoreiden kanssa sass, scss. Olen oppinut rakennuspaikkojen periaatteet gulp, myös perehtynyt versioohjattu git.

✅Mitkä minun suunnitelmat?
Jatka harjoittelua, Java Script, React, englanti. Etsia työ, jossa voin parantaa taitojani, osallistua erilaisiin mielenkiintoisiin hankkeisiin ja olla hyödyllinen ja myötävaikuteinen yrityksen kehitykseen.

✅Millaista koulutusta?
👨‍🎓 Valmistuin Pietarin valtion meritekniikan yliopistosta erikoisekonomisti-johtajana, taloustieteellinen tiedekunta, erikoistuminen - taloustiede ja johtaminen koneenrakennusyrityksessä. Viimeiset 5 vuotta olen asunut Suomessa. Olen Suomen kansalainen. 🇫🇮/🇷🇺.
